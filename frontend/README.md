# Todo App Frontend

A modern, responsive Todo application frontend built with Next.js, TypeScript, and Tailwind CSS featuring glassmorphism design.

## Features

- User authentication (login/register)
- Task management (create, read, update, delete)
- Responsive design for all device sizes
- Modern glassmorphism UI
- Real-time task synchronization
- Dark/light theme support

## API Integration

This frontend connects to the Todo App API backend. Make sure to configure the API endpoint in environment variables.

## Environment Variables

- `NEXT_PUBLIC_API_BASE_URL`: Base URL for the API backend
- `NEXT_PUBLIC_BETTER_AUTH_URL`: URL for authentication service
